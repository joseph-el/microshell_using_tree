# microshell_using_tree
Microshell is a small lightweight command-line interface or shell with a minimal set of features and commands.

## GRAMMAR
```Shell
FG           ::= 
                  | ";"

pipeline     ::=  command
                  |   pipeline "|" command

commandexec  ::=  word
```
